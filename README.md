# Game Store Hexlet

Полноценный интернет-магазин игр на Flask + SQLite + SQLAlchemy.

## Возможности

- 100+ популярных реальных игр создаются автоматически при первом запуске
- Регистрация, логин/логаут, профиль пользователя
- Баланс пользователя в рублях и пополнение баланса
- Корзина и покупка игр
- Страница игры с полной информацией
- Админ-панель для управления играми и просмотра профилей пользователей с их покупками

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python app.py
```

Приложение запускается на порту `5000`:

- http://127.0.0.1:5000

## Данные администратора

- Логин: `admin`
- Пароль: `admin123456`

Администратор создается автоматически при первом запуске.
